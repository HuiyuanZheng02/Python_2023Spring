4. 按照顺序执行如下语句，请给出每条语句执行后相应的问题的答案
x = y = z = 3000  # 请问x, y, z的取值是多少，有哪些变量指向同一个对象?
y = z + 2000      # 请问x, y, z的取值是多少，有哪些变量指向同一个对象?
x = x + 2000      # 请问x, y, z的取值是多少，有哪些变量指向同一个对象?
z = y             # 请问x, y, z的取值是多少，有哪些变量指向同一个对象?

答：
 1. 考察链式赋值语句以及赋值语句右边也是变量时，相当于为右边变量指向的对象取另一个别名
 因此x,y,z指向同一个整数对象3000, 都等于3000
 2. 考察赋值语句的计算顺序，首先右边计算得到5000，然后赋值给y。而原来x/z都不变，因此：
   x和z的值为3000， y的值为5000， x, z 指向同一个对象，即整数对象3000
 3. 类似的，右边计算出来为5000，然后x指向5000。注意右边并不是变量，因此x指向计算出来的整数对象5000，这样：
 x和y的值为5000, z的值为3000，x,y,z都指向不同的对象
 4. z = y , z指向y所指向的对象，因此x/y/z的值都为5000，而y和z指向同一个对象, x指向另一个对象
 

5. 判断下列名字哪些是非法的变量名，哪些虽然合法但不是一个好的变量名。
   (a)  class              (b) type         (c)  type_              (d) _
   (e)  mike's             (f) new-item     (g) 4bags               (h) student_id

   a. 非法的变量名：
   b. 合法但不建议的变量名：

答:

   a. 非法的变量名： class   mike's new-item 4bags
   b. 合法但不建议的变量名：type

   非法的变量名中 class是python关键字，mike's new-item都包含了标点符号，4bags最前面为数字，只允许字母下划线，后面可以是字母下划线数字
   type虽然合法，但是是内置函数，不建议使用
   type_ 合法，在实践中想要使用type作为名字但是由于为内置函数名，可以加个后缀_
   _ 也是合法的变量名，一般用于临时使用
   student_id 是合法的变量名
